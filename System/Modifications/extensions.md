#### This is the custom LuminOS and SDK extensions
okay, here the things become interessing, look,  
first, Linux kernel read the actual file type by the file header, NOT by the extension, so, every app extension is as application that can run in any linux distro  
custom extensions is made to organize the files.

.sdkx : official system LuminSDK executable<br>
.runx : applications made using LuminSDK or for LuminOS<br>
.photon : applications made with C++ and phantom lib<br>
.sln : this is just... Visual Studio project, thats almost useless, but it will be used in some builds =3<br>

these app extensions will appear on LuminOS CodeName "Eclipse" build 2097 // HEY i had to delay that feature to LuminOS CodeName "Photon" build 3020, due some problems compiling build 2097, this new codename means that we reached a new milestone (Photon) because of Kubuntu 26.04 LTS coming out in april.
